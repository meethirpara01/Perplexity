# Perplexity

A comprehensive, full-stack AI chat application built with modern web technologies that integrates multiple LLM providers with real-time internet search capabilities. The platform enables users to have intelligent conversations with AI assistants that can access current information, send emails, and provide context-aware responses.

Built a full-stack AI chat app using React, Node.js, and MongoDB with real-time messaging via Socket.io.
Integrated Gemini & Mistral AI using LangChain for intelligent, context-aware responses with internet search (Tavily).
Implemented JWT authentication, email verification, and secure REST APIs.
Developed responsive UI with Redux Toolkit & Tailwind CSS, including dark mode support.
